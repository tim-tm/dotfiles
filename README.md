# awesome dotfiles

Simple dotfiles in a gruvbox style that contain configurations for the following programs:

- AwesomeWM
- neovim
- polybar
- rofi
- kitty

## Dependencies

All of the above listed tools should of course be installed.
Additional:

- picom
- JetBrainsMono Nerd Font
- Gruvbox icon theme
- Noto Emoji Font
